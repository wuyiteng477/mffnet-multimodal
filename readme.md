Multimodal Data Processing for RGB–Fluorescence Tomato Seedling and Weed Detection

This repository provides the data preprocessing scripts used for RGB–fluorescence multimodal tomato seedling and weed detection.

The related study focuses on detecting tomato seedlings and co-occurring weeds using paired RGB images and fluorescence-enhanced grayscale images. Tomato seedlings were treated with carbon quantum dots to generate fluorescence responses under ultraviolet excitation, while RGB images provide visible colour, texture, morphology, and spatial structure information.

Note: This repository currently contains the dataset processing code only. The complete MFFNet training code, inference scripts, model configuration files, and trained weights will be released after the manuscript is accepted.
